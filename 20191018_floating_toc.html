目錄設置 (TOC)

pelicanconf setting

插件 : extract_toc

PLUGIN_PATHS = ['./pelican_plugins']
PLUGINS = [
    'i18n_subsites',
    "pelican_cjk",
    "pelican_cover_image",
    'neighbors',   
    'extract_toc',
    ]

Markdown

雖然如果不加上上面的extract_toc插件，直接放上下面的段落、也不修改HTML模板。

依然會在markdown中寫著[TOC]的區域，自動抓標題生成目錄。但若標題太多，導致版面過長反而不好看。

以下設置的'title'會包在 class=toc裡面。因為後面需要保留這個標題，所以我把從.py設定中拿掉，直接寫在HTML模板。

MARKDOWN = {
  'extension_configs': {
    'markdown.extensions.toc': {
      'title': '' ,
    },
    'markdown.extensions.codehilite': {'css_class': 'highlight'},
    'markdown.extensions.extra': {},
    'markdown.extensions.meta': {},
    # 'markdown.extensions.tables': { }, # 表格
  },
  'output_format': 'html5',
}

HTML template setting

根據官方文件，在HTML模板裡面想放置目錄的地方插入程式碼，以下是我的網頁模板片段。

<main class="content" role="main">
        <article class="post">
            {% if article.toc %}
                <div class="right-info" id="Toc">
                    <span class="toctitle">目錄</span>
                    {{ article.toc }}           
                </div>
            {% endif %}
        <div class="inner">
            <section class="post-content">
                {{article.content}}
            </section>

經過上面這樣的設定，我的 CSS才成功的抓到 class toc 套用變化 (markdown生成的目錄，預設是用<div class='toc'></div>包起來的 )。

CSS設定

主要參考文章 * 教學|HTML+CSS 加入浮動式自訂連結區塊 * 教學|CSS 網頁互動的好幫手 - hover的應用 * 簡易的沙盒 (可以現場實驗一些位置的屬性)

.right-info{
    max-width: 25%; 
    max-height: 50%;
    font-size:0.75em;
    top: 50%; /*數字越小越上面*/
    position: fixed;  /*固定在網頁上不隨卷軸移動 fixed ，若要隨卷軸移動用absolute*/   
    background: #aa99aa;  /*背景顏色*/
    padding: 10px 20px;
    border-radius: 10px;  /*圓角*/
    -moz-border-radius: 10px;
    -webkit-border-radius: 10px;    
}
.right-info .toc{ /*普通情況下，不要展開toc的內容*/
    display: none;
}
.toc {
    text-align: left; /*文字靠右*/
    line-height: 1em; /* 設定文字行距 */
    letter-spacing: 0.025em; /* 設定文字間距 */

}
.toc ul{ /*讓目錄的間隔緊實一點*/
    margin: 0px; 
    /*這個屬性是清單的'*'字號的縮排，預設(文章中)是40px 放在縮排目錄太寬*/
    padding-inline-start: 1em;
}
.right-info:hover{ /*當滑鼠碰到目錄框，展開內容。*/
    /*如果內容溢出，開啟滾輪(scroll)。
      使用滾輪模式需要設定母框架的大小(max-width/height)*/
    max-width: 25%; 
    max-height: 50%;
    overflow: scroll;
}
.right-info:hover .toc{  /*當滑鼠移至此區塊時，顯示toc內容*/
    display: block; 
}

其實我挺想嘗試position:sticky屬性 (想做到不要讓懸浮目錄標題擋到封面的效果)，不過這似乎是一個實驗性質的屬性，屢屢沒有測試成功，因此最後作罷。參見 : MDN Web Docs : position
