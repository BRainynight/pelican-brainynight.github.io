format

不只"{}".format("something")這種可以使用，大括號裡面可以發揮功能。 如下，裡面指定序號，會對應format中填的第幾個值，如此一來像有重複填入的情況，format也可以不用重複給值。

print('{0}, {1}, {2}'.format('a', 'b', 'c')) #a, b, c
print('{0}, {1}, {0}'.format('a', 'b')) #a, b, a
