前言

Pelican只是一個「生成靜態網站」的模組。預設上，輸出的地方是資料夾output.
這個output就是一個完整的網站，也是在本地 Demo 時的文件來源。把這些放上github，就是可以線上瀏覽的網站了。

連結github

我這裡出了點錯，可以說是讓output 下面又多了一層，不過就將就吧。 主要參考 Git 版本控制筆記 - 使用 github 及 ssh 金鑰設定

第一次建立遠端

git remote add (本地資料夾名稱) (完成SSH設定後，綠色Clone按鈕下得到的網址 git@github.com:XXXX/XXXX.github.io.git)

輸出資料夾: output/usrname.github.io，在這個資料夾 

git init #建立"監視系統"

之後作變更後，

pelican content #在主資料夾(ablog)下動作，生成新的網頁到指定輸出資料夾。

CMD進入輸出資料夾

git add .
git commit -m"本次更新內容，可自行改動敘述文字"
git push origin master

就會推送到網路上了。

ERROR

  1. fatal: not a git repository (or any of the parent directories): .git

    執行 git init
