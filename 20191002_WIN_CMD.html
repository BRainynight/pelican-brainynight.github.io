改變CMD開啟位置

  • 在指定目錄開啟CMD
    開啟的同時CMD工作位置即在當前資料夾:依靠滑鼠右鍵開啟
  • 開啟CMD立刻進入指定目錄
    執行 cmd 之後自動進入到指定的目錄:直接從捷徑更改 
# 進入C槽目錄
C:\Windows\System32\cmd.exe /K CD \

# 切到D槽就好
C:\Windows\System32\cmd.exe /K D:

# 進到C槽子目路
C:\Windows\System32\cmd.exe /K CD /D C:\Windows\Temp

# 進入D槽子目錄
C:\Windows\System32\cmd.exe /K CD /D "D:\Program Files"
