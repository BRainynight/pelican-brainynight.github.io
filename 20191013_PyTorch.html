屬性( attributes)

  1. dtype : 如32-bit floating等

  2. device : torch.Tensor被分配的設備類型 : "cpu" / "cuda"
    cpu 和 cuda 的轉換，靠 .to實踐

    python data = torch.Tensor([1]) data.to(device_cpu) 3. layout : 表內存分布，目前只支持 torch.strided
